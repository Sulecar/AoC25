# Day06

Prvi del naloge se osredotoča na ekstrakcijo števil iz vrstic vhodne datoteke znotraj zaznanih meja posameznega problema. Program najprej identificira stolpce, ki ločujejo različne bloke podatkov, nato pa v vsakem bloku prebere števila, ki so zapisana v vodoravni smeri. Za vsak sklop števil se iz zadnje vrstice prebere pripadajoča matematična operacija, bodisi seštevanje bodisi množenje. Funkcija nato zaporedno izvede to operacijo nad vsemi pridobljenimi števili in rezultat prišteje h končni vsoti prvega dela. Celoten postopek temelji na predpostavki, da posamezno število predstavlja celotna vrstica znotraj določenega razpona stolpcev.

Drugi del naloge uporablja isto logiko zaznavanja meja problemov in operacij, vendar spremeni način branja števil. Namesto branja po vrsticah program pregleda vsak stolpec posebej in sestavi števila iz števk, ki so zapisane navpično skozi vrstice. Takšen pristop omogoča obdelavo podatkov, kjer so vrednosti orientirane drugače kot v prvem delu. Ko so vsa navpična števila zbrana, se nad njimi ponovno izvede določena operacija seštevanja ali množenja. Končni izračun se nato shrani kot ločen rezultat, ki predstavlja rešitev drugega dela problema.
