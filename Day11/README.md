# Day11

Program najprej iz vhodne datoteke zgradi usmerjen graf v obliki seznama sosedov. Z uporabo rekurzivne funkcije za iskanje v globino (DFS) sistematično raziskuje vse poti od vozlišča "you" do cilja "out". Varnostni mehanizem sproti beleži obiskana vozlišča, kar preprečuje neskončne cikle in zagotavlja pravilno štetje unikatnih poti.
